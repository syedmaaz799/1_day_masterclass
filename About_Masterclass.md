## 2. What this masterclass is

This is a **live, hands-on build session** — not a lecture series, not a recorded course, not a theory workshop.

Attendees join online for **two afternoons** and **build real AI agents from scratch** alongside the instructor. Every session ends with something **working and published** — not slides, not mockups, not “coming soon” demos.

The masterclass sits at the intersection of:

- **Agentic AI** — systems that think, decide, route tasks, and act
- **No-code production** — the same visual approach teams use to ship workflows without writing application code
- **Modern developer literacy** — GitHub, IDEs, and app builders introduced on Day 2 so builds become shippable products

The emotional arc for attendees: **curiosity → capability → ownership**. They arrive wondering if they can build with AI. They leave with four live agents they built themselves.

## 5. What attendees will build (four agents — full detail)

Use this section for product shots, feature callouts, before/after visuals, and build montages in video.

---

### Agent 1 — Q&A Chatbot

**What it is:** The attendee’s first live AI agent — the foundation every other build extends.

**How it works:**

- User input goes in
- A language model processes the request
- A clear answer comes out

**What it teaches:**

- The **three-step pipeline** every agent runs on: input → model → output
- How to wire a simple agent and **publish it live**
- The mental model of an AI employee as a **system**, not a chat window

**Creative angle:** “Your first AI employee — live in one hour.”

---

### Agent 2 — AI Content Creator Assistant

**What it is:** An **intent-routing content engine** — one assistant that understands what the user wants and generates the right type of content.

**How it works:**

- The system **classifies the user’s intent** (what kind of content they need)
- Routes to the correct generation path
- Returns **publish-ready output**

**Two main content paths:**

| Path | What it produces |
|------|------------------|
| **Social posts** | Hooks, CTAs, hashtags — ready to post |
| **Ad copy** | Headlines, emotional triggers — ready to run |

**Five specialized paths total** — one assistant, multiple specialized outputs.

**What it teaches:**

- **Intent classification** — how agents decide what to do before they do it
- **Routing logic** — one entry point, many specialized outcomes
- Building agents that **replace repetitive creative work**

**Creative angle:** “One assistant. Five paths. Content on demand.”

---

### Agent 3 — AI Resume Reviewer

**What it is:** A document-intelligence agent that reads CVs and returns actionable career output.

**How it works:**

- User **uploads a CV**
- The system **extracts** the document
- **Analyzes** each section
- **Routes** the request to the right output type
- Returns **improvement suggestions** the user can apply immediately

**Five request types it handles:**

1. Full resume review  
2. Resume summary  
3. Cover letter generation  
4. Interview preparation  
5. Career advice  

**What it teaches:**

- **Document extraction and analysis** in an agent pipeline
- **Multi-path routing** based on user intent
- Building agents that handle **real professional tasks**

**Creative angle:** “Upload a CV. Get career intelligence back.”

---

### Agent 4 — Voice News Intelligence Briefing

**What it is:** A **voice-first research app** — not a chatbot, a product.

**How it works:**

- Scans news sources
- **Synthesizes** what matters into a clear briefing
- **Speaks** the briefing back to the user

**What it teaches:**

- Building a **voice-first AI product** from scratch
- News research and synthesis pipelines
- Taking a build from **idea → app → live deployment**

**Creative angle:** “Your personal news analyst — built and shipped in one session.”

## 10. The bigger picture — why agents matter now

Use this for opening hooks, problem-agitation in ads, and narrative voiceover.

### The shift

Work is moving from **manual repetition** to **agent-driven automation**. An AI employee does not just answer — it **acts**:

| Task | Manual today | With an agent |
|------|--------------|---------------|
| Lead qualification | Skim every inquiry and guess who is worth a reply | Score and prioritise each lead the moment it lands |
| Email drafting | Rewrite the same reply for the hundredth time | A tailored draft ready before you open the inbox |
| Follow-ups | Forget who to chase, and when | Scheduled and sent on time, automatically |
| Customer support | Repetitive questions eat your afternoons | Routine answers handled; complex cases escalated |
| Scheduling | Six messages to land on one meeting slot | Slot booked and invite sent |
| Research | Twenty tabs to brief a single prospect | Concise, sourced summary in seconds |

**Payoff line:** *Your agent can do it all automatically.*

### The AI timeline (context for scripts)

- **2023** — ChatGPT: AI as conversation  
- **2024** — Assistants: AI with tools  
- **2025** — Workflows: AI that chains tasks  
- **2026** — AI Employees: AI that runs business processes  

Attendees are at the edge of this curve. This masterclass is where they **build**, not just watch.

### The companies building the stack (context only)

Attendees gain literacy in the ecosystem powering agentic AI — OpenAI, Google, Anthropic, NVIDIA, Microsoft, Meta, Perplexity, Groq, Hugging Face, LangChain, and others. This context is woven into the masterclass narrative; it is not a separate lecture block.
